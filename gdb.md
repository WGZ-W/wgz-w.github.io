## 开始调试
```
(gdb) start     从头开始调试 

(gdb) r         开始运行到第一个断点
```

## 退出正在调试的函数
```
(gdb) finish    函数继续执行完，打印返回值

(gdb) return    不继续执行，直接返回
(gdb) return exp    修改函数的返回值
```

## 打印变量值
```
(gdb) p [value]     打印内置变量的值

(gdb) p 
```

## 执行下一句
```
(gdb) n     执行当前语句，函数调用不进入函数

(gdb) s     执行当前语句，若是调用函数则进入函数
```

## 设置断点
```
(gdb) b [line]      在 line 行设置断点

(gdb) i b           查看所有断点

(gdb) d [number]    删除编号为 number 的断点
```
