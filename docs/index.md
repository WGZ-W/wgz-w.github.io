- 12：00-13：00 午饭时间
- 13：00-14：00 午休
- 14：00-17：00 下午的工作时间
- 17：00-19：00 健身+晚饭
- 19：00-21：00 晚上的工作时间
- 21：00-23：00 洗漱，洗衣服，收拾东西，视频聊天

- 读文献
- 写代码

### 午休
中午午休半小时，起床后用洗面奶洗一下。
