


# 公考复习计划

## 三阶段复习总览
关键在于**完成比完美更重要**，行动起来才最有用！

|阶段|时间|核心目标|
|-|-|-|
|基础入门|1-3个月|听课+刷基础题，建立知识体系|
|强化提升|1-2个月|分模块刷题，提速提准，攻克弱项|
|冲刺模考|考前1-2个月|做真题套卷，模拟考场，调整节奏|


## 行测：分模块攻克
行测是速度与正确率的博弈，每个模块都有提分技巧。

- **数量关系**
- **政治理论**
- **资料分析：** 这是性价比最高的模块，一定要拿下！第12篇 推荐跟 花生十三或 高照 打基础，掌握速算口诀，目标是做到20题全对。
- **判断推理：** 逻辑和图推是难点。逻辑判断跟花生十三，思路清晰；图形推理看 龙飞 ，他能帮你站在出题人角度思考。 类比推理可以听听 聂佳 的短课，效率很高。
- **言语理解：** 主要靠语感和技巧。推荐 郭熙或 阿里木江 ，学着抓主题词和文段结构，每天保持一定题量。 逻辑填空要积累，可以刷刷 花生的高频700词。
- **常识判断：** 平时积累最重要。推荐 李梦娇 的速记口诀，利用碎片时间听 喜马拉雅 的时政，或者看《人民日报》、《半月谈 》的评论文章。


## 申论：动笔才是硬道理
申论不是光听课就能学会的，核心在于“动笔+总结”。

- 小题： 学会从材料里“抄”关键词。可以跟 袁东或 站长 的课，把一道题的答案写两遍，一遍自己写，一遍对照讲解优化，吃透答题逻辑。
- 大作文： 每周至少写一篇。重点不是背模板，而是把看到的好词好句、政策理论、人民日报的评论段落摘抄下来，内化成自己的东西。


## 备考工具箱

- 题册： 《行测5000题 》是刷题神器，申论可以搭配《申论100题》。 
- APP： **B站** 搜免费课，**公考雷达** 查岗位，**番茄TODO** 做计划本。



## 上岸心态指南

- 专注当下： 别总想“等我把XX模块学完再开始刷题”，学完一个模块就立刻刷题巩固，效率最高。
- 错题为王： 错题当天整理，用不同颜色标记原因，复盘的时间可能比做题还长，但这是提分的关键。
- 模拟考场： 严格按照考试时间（上午9-11点行测，下午申论）做套题，把手机关机，全身心投入，找到考试的感觉。
