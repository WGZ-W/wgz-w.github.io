# Basic

## 正确的关机
查看目前在线上的人： 
```
$ who
```
而要看背景执行的程序可以使用：
```
ps -aux
```
重启：
```
$ sudo reboot
```

## 磁盘与目录容量
磁盘：df（disk free）
```
$ df -h
```
目录：
```
$ du -sh
```

## 命令行热键
- [Tab]，补全
- [Ctrl]-c，命令取消
- [Ctrl]-d，对话结束
- [shift]+{[PageUP]|[Page Down]}，翻页
